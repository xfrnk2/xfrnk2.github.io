정렬 알고리즘 - 버블 정렬(Bubble Sort) 정리

2020-07-10 Algorithm xfrnk2

버블정렬의 원리와 특징 및 성능에 대해서 정리하는 글이다.

학습 목표
버블정렬의 원리와 특징 및 성능에 대해서 설명할 수 있다.
버블정렬을 프로그래밍 언어로 구현할 수 있다.

버블정렬의 원리

  • 모든 인접한 값의 비교를 위해 왼쪽 또는 오른쪽에서 시작하여 반대편 끝으로 이동
  • 인접한 두 값을 비교하여 왼쪽 값이 더 큰 경우 자리를 바꾸는 과정을 반복

버블정렬의 정렬 과정

  • 배열 [7, 3, 5, 8, 4]이 주어진 경우
  • 아래의 예시에선 왼쪽에서 오른쪽으로 정렬을 수행
  • 비교연산의 기준이 되는 시작하는 위치는 가장 끝에서 가장 끝 바로 하나 전의 위치까지, 진행 방향으로 1씩 이동

초기 상태

[7, 3, 5, 8, 4]
  1. 1번째 위치에서 시작
  • 7과 3의 교환 -> [3, 7, 5, 8, 4] (1, 2번째 값 비교)
  • 7과 5의 교환 -> [3, 5, 7, 8, 4] (2, 3번째 값 비교)
  • 3, 4번째 값 비교시에는 교환이 일어나지 않는다
  • 8과 4의 교환 -> [3, 5, 7, 4, 8] (4, 5번째 값 비교)

다음 상태

[3, 5, 7, 4, 8]
  1. 2번째 위치에서 시작
  • 2, 3번째 값 비교시에는 교환이 일어나지 않는다
  • 5과 4의 교환 -> [3, 5, 4, 7, 8] (3, 4번째 값 비교)
  • 4, 5번째 값 비교시에는 교환이 일어나지 않는다

다음 상태

[3, 5, 4, 7, 8]
  1. 3번째 위치에서 시작
  • 3, 4번째 값 비교시에는 교환이 일어나지 않는다
  • 4, 5번째 값 비교시에는 교환이 일어나지 않는다
[3, 4, 5, 7, 8]
  1. 4번째 위치에서 시작
  • 4, 5번째 값 비교시에는 교환이 일어나지 않는다

버블정렬의 Python 코드

array = [20, 60, 10, 70, 30, 50, 40, 80]
n = len(array) # array의 길이(크기)

for i in range(n-1):
    for j in range(i+1, n):
	   if array[i] > array[j]:
           array[i], array[j] = array[j], array[i]
