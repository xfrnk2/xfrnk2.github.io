프로그래머스-해시-완주하지 못한 선수로 이동

첫 성공시도

def solution(participant, completion):
    answer = ""
    counts = dict()
    for i in completion:
        counts[i] = counts.get(i, 0) + 1
    for name in participant:
        value = counts.get(name)
        if value == None or value <= 0:
            answer = name
            break
        counts[name] -= 1
    return answer

셀프 피드백

  • 복잡하다.
  • 더 간단히 할 수는 없을까?

두번째 성공시도

from collections import Counter

def solution(participant, completion):
    i = iter((Counter(participant) - Counter(completion)).keys())
    return next(i)

간단 메모

  • collections의 Counter가 뺄셈 연산이 가능한 점이 새로웠다.
  • iter()함수로 iterable하게 만든 후, next()를 통해 첫번째 값을 얻었다.
