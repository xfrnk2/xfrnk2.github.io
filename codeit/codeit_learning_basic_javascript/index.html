코드잇 - 자바스크립트 기초 공부

2021-07-22 codeit xfrnk2

요즘은 배웠던 자바스크립트, 그리고 HTML 과 CSS를 가지고 웹 페이지를 만들어서 호스팅 후 확인해보는 것을 일삼으며 지내곤 하는데, 시간이 날 때마다 자바스크립트 기초 강의를 클릭, 클릭, 코딩, 코딩.. 하다 보니 빠르진 않으나 어느덧 절반 가까이 수강했다.

지난 수년간 파이썬을 가지고 이것 저것 만들어 보면서 프로그래밍의 기본은 알고 있었지만, 다른 도구를 손에 잡으니 어딘가 2% 부족하고, 잘 모르는 듯한 느낌을 지울 수가 없었다. 그 주인공이 바로 자바스크립트.

​

강의내용은 사실 쉬운 편에 속하는 것 같다. 파이썬도 프로그래밍에 입문하기 좋은 언어라지만, 자바스크립트 기초를 배울 수 있도록 만들어진 본 강좌도 이에 뒤쳐지지 않을 만큼 굉장히 이해하기 쉽고, 체득해 가기 좋은 구조를 가지고 있다.

현재 캡쳐한 사진의 챕터 이름은 무려 ‘함수 부려먹기’… 더이상의 자세한 설명은 안하는 것으로..

​

아무튼 기초적이고 작은 부분부터라도 시간을 내서 듣다 보니, 이전에 사용하던 파이썬과 다르게, “아, 파이썬은 이렇게 됬었지만, 자바스크립트는 또다른 형태과 구조를 가지는구나!“하고 차이점을 이따금 발견하곤 한다.

아무튼 요즘 코드잇 강의를 수강하는 일 보다는, 기존의 지식 + 자바스크립트 기초 강의를 들으며 직접적으로 웹 사이트를 만들어 보면서 활용하며 컴퓨터 앞에 있는 일이 부지기수인 것 같다.

​사실 웹 사이트를 만들다보니, 자바스크립트 또한 중요하지만, 어느정도 맥락을 잡은 시점에서 필요한 것 위주로 정독하고 빠르게 넘어가면서, SQL과 서버딴 언어(유닉스 커맨드라인)를 조금 더 집중도 있게 배우는 편이 좋지 않을까 하는 생각이 드는 오늘이다.

​프로그래밍 입문을 하고자 하시는 분 중에, 나는 나만의 웹 사이트를 만들어 보고 싶다는 생각을 가지고 계신 분께서는 파이썬도 좋지만 자바스크립트 기초 강의를 한 번 수강 시작해보신다면 좋을 것 같다.

​손을 뻗으면 닿을 곳에 나의 기초를 탄탄하게 해줄 수 있는 강의들이 포진해 있어서 다행스럽고도 고마운 일이다.

​

앞으로도 최대한 시간을 내서 수강해 가보자!!
