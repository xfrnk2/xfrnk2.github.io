Python - isort Error

2021-04-05 resolved problem xfrnk2

문제 발생 상황

  • https://pypi.org/project/isort/
  • isort는 python에서 import하는 라이브러리들을 자동으로 정렬해준다.
isort file.py
  • 이후 나오는 질문에선 y를 입력하면 된다.
